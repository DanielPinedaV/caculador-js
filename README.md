# 🧮 Calculadora Moderna con JavaScript

> Una calculadora funcional, responsiva y con diseño moderno, construida con HTML, CSS y JavaScript puro (sin frameworks).

## ✨ Características

- ✅ Operaciones básicas: suma, resta, multiplicación y división
- ✅ Pantalla dinámica con visualización en tiempo real
- ✅ Botón de borrado total (`C`) y borrado parcial (`⌫`)
- ✅ Manejo de errores con mensaje "Error" en operaciones inválidas
- ✅ Diseño moderno y responsivo (CSS Grid/Flex, variables CSS, animaciones suaves)
- ✅ Compatible con escritorio y móvil
- ✅ Código limpio, comentado y fácil de mantener

---

## 🚀 Tecnologías utilizadas

| Tecnología | Propósito |
|------------|-----------|
| **HTML5** | Estructura semántica de la interfaz |
| **CSS3** | Estilos modernos, variables CSS, media queries, animaciones |
| **JavaScript (ES6+)** | Lógica de operaciones, manipulación del DOM, manejo de eventos |
